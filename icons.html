ui-icons for Fieldbook

These icons have been saved in all states & colors that might be needed. The icons are built in Adobe Illustrator CS5 on a 512x512px document, then exported as transparent pngs. The file structure is color name _ color code. So black would be black_000.png. The icon naming convention is thus: ui-icon denotes that the class is an icon. This is followed by a detailed description of the icon using dashes for spaces. So ui-icon-folder-open tells us that we are dealing with an icon that is an open folder. Icons will fall into groups. We need to keep groups in mind as we add icons. This will help us stay organized, stop icon soup, & help us remember what, where & how to use all of our icons before we make new ones.
